# AI Interview Preparation App

A full-stack web application that generates personalized interview preparation plans based on company, role, experience level, and interview type.

## Tech Stack
- Frontend: Next.js (React + TypeScript)
- Backend: FastAPI (Python)
- API communication: REST

## Features
- Select company (Amazon, Google, Microsoft)
- Choose role and experience level
- Generate customized prep plans
- Covers:
  - Data Structures
  - Algorithms
  - Behavioral Questions
  - System Design

## Run the Project

### Backend
cd backend  
uvicorn main:app --reload

### Frontend
cd frontend  
npm install  
npm run dev

Open:
http://localhost:3000
